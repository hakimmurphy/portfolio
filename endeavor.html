Endeavor Music

A music label website

Living on heroku, this is a record label full stack application I created using HTML, CSS, JavaScript, Node, EJS, Mongo, Mongoose, and JOI. I showcase my understanding of both frontend technologies and backend methods to build a place where users can register and login create, edit, and delete posts.

Using vanilla JavaScript, I make dynamic buttons that animate and toggle text. I specifically did not use any JS libraries in order to practice learning logic and DOM manipulation.

I made a place for posts where a user can upload texts and photos. So at the bottom of the index.html is where the posts are posted.

Below is the post page that is clicked, and shows all the individual data the user has inputed.

Created a form from scratch to practice my CSS skills that applies to both the register and login routes.
